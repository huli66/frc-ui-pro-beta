# ConfigProvider全局化配置

为组件提供统一的全局化配置。(多语言)
    
> 详细文档，请参阅 antd -> ConfigProvider全局化配置 -> 地址：https://ant.design/components/config-provider-cn/

## 使用

ConfigProvider 使用 React 的 context 特性，只需在应用外围包裹一次即可全局生效。

```ts
import { ConfigProvider } from 'antd';

// ...

export default () => (
  <ConfigProvider direction="rtl">
    <App />
  </ConfigProvider>
);
```
